Edx_Custom_Themes

Edx平台初步设置

  1. virusalbox ubuntu 模式，网络选择NAT模式；
  2. sudo vi /etc/network/interfaces
  3. 把网络地址部分全部都注释掉即：

    auto eth0；#此处eth0在ubuntu16.04中为enp0s17,利用ifconig -a可以查询；
    iface enp0s17 inet dhcp #此处由static 改为 dhcp；
    注释掉静态地址和子网掩码；

  4. 网站初始化设置：

    学生界面http://your_ip/;
    教师课程内容编辑界面http://your_ip:18010;
    教师账号：staff@example.com / edx;
    verified@example.com / edx;
    audit@example.com / edx;
    学生账号：honor@example.com / edx;

    Theme目录结构:

    /my-open-edx-themes/edx-platform/my-theme/lms/static/images
    /my-open-edx-themes/edx-platform/my-theme/lms/static/sass
    /my-open-edx-themes/edx-platform/my-theme/lms/templates

    • sudo chown -R edxapp:edxapp /my-open-edx-themes #权限赋予edxapp用户
    • sudo chmod -R u+rw /my-open-edx-themes
    • /edx/app/edxapp/lms.env.json 设置 “ENABLE_COMPREHENSIVE_THEMING”: true
      LMS, add the path to COMPREHENSIVE_THEME_DIRS in /edx/app/edxapp/lms.env.json. "COMPREHENSIVE_THEME_DIRS": [ "/my-open-edx-themes/edx-platform" ]

      如何使用edx-bootstrap-theme

  5. sudo apt-get install git #安装git;
  6. git clone https://github.com/dadasoz/edx-bootstrap-theme.git
  7. sudo apt-get install python-paver #有时候需要安装这个程序；
  8. 打开’/edx/app/edxapp/lms.env.json’ and edit COMPREHENSIVE_THEME_DIR to /edx/app/edxapp/edx-platform/themes/edx-bootstrap-theme #此处填写实际了路径；

  9. cd /edx/app/edxapp/edx-platform；
    sudo -H -u edxapp bash
    source /edx/app/edxapp/edxapp_env #重要步骤不能缺少；
    cd /edx/app/edxapp/edx-platform
    paver update_assets cms –settings=aws
    paver update_assets lms –settings=aws
    sudo /edx/bin/supervisorctl restart edxapp:
    sudo /edx/bin/supervisorctl restart edxapp_worker:
