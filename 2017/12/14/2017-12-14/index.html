EDX自定义Theme

2017-12-14
edx

Edx平台初步设置

  1. virusalbox ubuntu 模式，网络选择NAT模式；
  2. sudo vi /etc/network/interfaces
  3. 把网络地址部分全部都注释掉即：

    auto eth0；#此处eth0在ubuntu16.04中为 enp0s17,利用ifconig -a可以查询；
    iface enp0s17 inet dhcp #此处由static 改为 dhcp；
    注释掉静态地址和子网掩码；

  4. 网站初始化设置：

    学生界面 http://your_ip/;
    教师课程内容编辑界面 http://your_ip:18010;
    教师账号：staff@example.com / edx;
    verified@example.com / edx;
    audit@example.com / edx;
    学生账号：honor@example.com / edx;

Theme目录结构:

/haue-theme/edx-platform/zyhj-theme/lms/static/images
/haue-theme/edx-platform/zyhj-theme/lms/static/sass
/haue-theme/edx-platform/zyhj-theme/lms/templates

  • sudo chown -R edxapp:edxapp /my-open-edx-themes #权限赋予edxapp用户
  • sudo chmod -R u+rw /my-open-edx-themes
    /edx/app/edxapp/lms.env.json 设置 “ENABLE_COMPREHENSIVE_THEMING”: true
    LMS, add the path to COMPREHENSIVE_THEME_DIRS in /edx/app/edxapp/lms.env.json.
    “COMPREHENSIVE_THEME_DIRS”: [“/haue-theme/edx-platform”]
    重启所由服务；

设置应用Theme

登陆django管理后台；#http://your_url/admin
选择Site themes;
从Site选择需要应用theme的网站;
在Theme dir name填入theme名称；#例如zyhj-theme

编译Theme

cd /edx/app/edxapp/edx-platform；
sudo -H -u edxapp bash
source /edx/app/edxapp/edxapp_env #重要步骤不能缺少；
cd /edx/app/edxapp/edx-platform
paver update_assets lms –settings=aws
sudo /edx/bin/supervisorctl restart edxapp:
sudo /edx/bin/supervisorctl restart edxapp_worker:

参考

[ Changing Themes for an Open edX Site] (http://edx.readthedocs.io/projects/edx-installing-configuring-and-running/en/open-release-ginkgo.master/configuration/changing_appearance/theming/index.html)
